# Solution: Mainframe Data Manipulation with Python

This directory contains the completed version of the workshop.

## How to Run

1.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

2.  Start the service:
    ```bash
    python manipulator_service.py
    ```

3.  In another terminal, run the client:
    ```bash
    python client.py
    ```

4.  Enter your mainframe credentials to trigger the manipulation process.
